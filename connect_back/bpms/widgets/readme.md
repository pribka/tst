#### Копирование настроек виджетов с рабочего стола пользователя

**Endpoint:** `POST /api/v1/widgets/desktop_templates/copy-from-desktop/`

**Описание:** Позволяет администраторам копировать настройки виджетов с рабочего стола пользователя в шаблон для новых пользователей.

**Требования:**
- Только суперпользователи могут выполнять эту операцию
- Пользователь должен иметь настроенный рабочий стол

**Request Body:**
```json
{
    "desktop_template": "863caee4-718e-11ee-8c1b-4216f3de51df"
}
```

**Параметры:**
- `desktop_template` (string, required) - UUID рабочего стола пользователя (UserDesktopModel), настройки которого нужно скопировать
