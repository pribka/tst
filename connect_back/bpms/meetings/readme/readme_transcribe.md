# Эндпойнты для работы с транскрибацией видеозаписей

## 1. Получить список видеозаписей без транскрибации

**GET** `http://d.centersoft.kz:8000/api/v1/meetings/records/untranscribed/`

**Параметры запроса:**
- `ordering` - сортировка (например: `-created_at`)
- `page_size` - размер страницы (по умолчанию `1`, если не передан)
- `start` - фильтрация по начальной дате (формат: `2025-10-26T00:00:00.000Z`)
- `end` - фильтрация по конечной дате (формат: `2025-12-08T00:00:00.000Z`)

**Пример запроса:**
```
http://d.centersoft.kz:8000/api/v1/meetings/records/untranscribed/?ordering=-created_at
```

**Пример с фильтрацией по времени:**
```
http://d.centersoft.kz:8000/api/v1/meetings/records/untranscribed/?start=2025-10-26T00:00:00.000Z&end=2025-12-08T00:00:00.000Z
```

**Примечания:**
- Если не передан `page_size`, то выводится `page_size=1`
- По умолчанию выдаются записи после 15.11.2025, потому что с этого времени видеозапись привязывается к сессии
- До этого времени нет возможности определить фактическое количество участников
- Доступен только для суперпользователей (user.is_staff)

---

## 2. Изменить статус видеозаписи

**POST** `http://d.centersoft.kz:8000/api/v1/meetings/records/status/`

**Тело запроса:**
```json
{
  "id": "3fa9fc2e-c51a-11ec-9d2c-0242ac110007",
  "status": "processing"
}
```

**Все возможные статусы:**
- `new` (по умолчанию)
- `processing` - используется, пока проходит транскрибация (чтобы не запустить транскрибацию одной и той же видеозаписи два раза)
- `done` - присваивается, когда готова транскрибация
- `error`

**Примечания:**
- Доступен только для суперпользователей (user.is_staff)

---

## 3. Записать транскрибацию

**POST** `http://d.centersoft.kz:8000/api/v1/meetings/records/set_transcribe/`

**Тело запроса:**
```json
{
  "id": "3fa9fc2e-c51a-11ec-9d2c-0242ac110007",
  "text": "Текст транскрибации",
  "text_json": {},
  "summary": "Краткое саммари"
}
```

**Примечания:**
- При сохранении транскрибации автоматически устанавливается статус `done`
- Доступен только для суперпользователей (user.is_staff)
