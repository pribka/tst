# Icon cheat sheet — flaticon UIcons (regular rounded)

The project loads `uicons-regular-rounded.css` from `/src/assets/css/uicons-regular-rounded.css`.
For prototypes the same set is available via the public CDN:

```html
<link rel="stylesheet" href="https://cdn-uicons.flaticon.com/uicons-regular-rounded/css/uicons-regular-rounded.css">
```

Use icons as `<i class="fi fi-rr-<name>"></i>`.
