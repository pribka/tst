# Задания По Блоку: Основа Системы И Объекты Управления

## Цель

Собрать во фронте единый объектный контекст, чтобы любая работа в Connect была видима, связана с объектом управления и доступна через понятную навигацию.

## Задания

1. Ввести единый object context bar для карточек, форм и drawer-экранов.
   Модули: `Projects`, `Directories`, `HelpDesk`, `Orders`, `vue2TaskComponent`, `vue2MeetingComponent`, `Documents`.

2. Добавить обязательные связки с объектами управления в сценарии создания и редактирования.
   Нужно: унифицированные селекты `проект`, `контрагент`, `направление`, `подразделение`, `сотрудник` там, где это допустимо по модели.

3. Сделать reusable-компоненты object chips, object links и object breadcrumbs.
   Модули: `UIModules`, общий слой компонентов.

4. Ввести фронтовые индикаторы полноты цифрового следа.
   Примеры: нет привязки к объекту, нет комментария-итога, нет файлов, нет трудозатрат, нет следующего шага.

5. Подготовить единый object timeline shell.
   Это должен быть переиспользуемый контейнер, куда потом встраиваются задачи, встречи, обращения, документы, комментарии и сигналы.

## Критерии приемки

- Пользователь из любого ключевого рабочего экрана видит, с каким объектом он сейчас работает.
- В новых и редактируемых сущностях объектные связи оформлены одинаково.
- У объекта появляется единый визуальный вход в историю действий и связанных артефактов.
