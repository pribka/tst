# Задания По Блоку: Паспорта Объектов, Аналитика И ИИ-Оценка

## Цель

Создать во фронте цифровые паспорта объектов управления и слой ИИ-оценки, который помогает быстро понять состояние проекта, клиента, контракта или направления.

## Задания

1. Спроектировать единый shell цифрового паспорта.
   Основные вкладки: обзор, финансы, активность, коммуникации, задачи, трудозатраты, документы, риски, ИИ-оценка.

2. Реализовать паспорт проекта как первый эталонный объект.
   Модули: `Projects`, `Reports`, `Dashboard`, `vue2TaskComponent`, `vue2MeetingComponent`.

3. Реализовать паспорт контрагента/клиента.
   Модули: `Directories`, `HelpDesk`, `Orders`, `Documents`.

4. Добавить AI assessment cards.
   Нужно: итоговая оценка, факторы оценки, динамика, проблемные зоны, переход к первичным данным.

5. Собрать cross-source виджеты для паспорта.
   Источники: задачи, обращения, встречи, коммуникации, трудозатраты, документы, отчеты.

6. Добавить drill-down и explainability.
   Пользователь должен понимать, откуда взялась каждая цифра и каждый ИИ-вывод.

## Критерии приемки

- Паспорт объекта дает руководителю и исполнителю целостную картину без ручного сбора данных.
- ИИ-оценка не выглядит “магией”, а объясняется через конкретные артефакты работы.
- Из паспорта можно быстро перейти к любому первичному источнику.
