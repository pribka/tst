# Задания По Блоку: Встречи, Звонки И Внутренние Коммуникации

## Цель

Превратить коммуникации в полноценный объект системы: связанный, анализируемый и конвертируемый в дальнейшую работу.

## Задания

1. Ввести единый communication timeline.
   Источники: встречи, звонки, чаты, комментарии, файлы, AI-саммари.

2. Реализовать отдельный фронтовый контур для звонков.
   Минимум: карточка звонка, участники, привязка, запись, итог, follow-up.

3. Добавить универсальные действия из коммуникаций.
   `создать задачу`, `создать встречу`, `создать заявку`, `добавить в паспорт объекта`.

4. Собрать обязательный post-meeting flow.
   Проверка: запись, резюме, намерения, ответственные, сроки, привязка к объектам.

5. Встроить AI-обработку коммуникаций в основной UI.
   Модули: `AIAssistant`, `vue2MeetingComponent`, `vue2ChatComponent`.

6. Добавить объектные и коммуникационные фильтры.
   Срезы: проект, клиент, сотрудник, команда, период, тип коммуникации.

## Критерии приемки

- Пользователь видит все связанные коммуникации по объекту в одном месте.
- После встречи или звонка система подталкивает к фиксации результата.
- Из коммуникации можно за один-два клика создать дальнейшие артефакты работы.
