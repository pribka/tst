# Задания По Блоку: Планирование, Задачи И Недельный Ритм

## Цель

Собрать во фронте единый недельный сценарий сотрудника: спланировал, выполнил, перепланировал, закрыл хвосты, обработал висяки.

## Задания

1. Сделать `weekly workspace` в `WorkPlan`.
   Состав: неделя, день, backlog, просрочка, висяки, календарные события, быстрые действия.

2. Ввести сценарий утреннего планирования.
   Нужно: пустое состояние, чек-лист “что нужно сделать перед началом дня/недели”, быстрые добавления задач и событий.

3. Реализовать отдельный UX статуса `висяк`.
   Нужно: причины, владелец решения, дата пересмотра, фильтры, счетчики, менеджерский обзор.

4. Добавить reason-capture для переноса срока и незавершенности задачи.
   Модули: `vue2TaskComponent`, `WorkPlan`, `Projects`.

5. Добавить виджеты качества планирования.
   Примеры: без срока, без объекта, просрочено, без трудозатрат, зависло, без следующего шага.

6. Встроить AI-перепланирование недели.
   Модули: `AIAssistant`, `WorkPlan`.

## Критерии приемки

- Пользователь может пройти неделю через один основной экран.
- Любой перенос и любой висяк оформляются явно.
- Руководитель видит хвосты и качество планирования команды без ручной сборки данных.
