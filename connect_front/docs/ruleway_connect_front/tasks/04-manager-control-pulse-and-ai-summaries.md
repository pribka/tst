# Задания По Блоку: Управленческий Контур, Пульс Дня И ИИ-Сводки

## Цель

Собрать менеджерский слой, который превращает разрозненные рабочие данные в сигналы, ежедневный контроль и понятные ИИ-сводки.

## Задания

1. Создать экран `Пульс дня`.
   Два режима: сотрудник и руководитель.

2. Реализовать ленту управленческих сигналов.
   Нужны: приоритет, источник сигнала, объект, срок реакции, статус обработки.

3. Собрать `manager cockpit`.
   Модули: `Dashboard`, `Reports`, `WorkPlan`, `Notifications`.

4. Выделить ежедневную ИИ-сводку в самостоятельный пользовательский объект.
   Нужно: история сводок, доставка, статус прочтения, переход к исходным данным.

5. Добавить объяснение ИИ-выводов.
   Пользователь должен видеть, из каких задач, обращений, встреч и трудозатрат собран вывод.

6. Встроить уведомления о критических сигналах.
   Модули: `Notifications`, `AIAssistant`.

## Критерии приемки

- Руководитель получает управленческую картину без ручного обхода нескольких экранов.
- Сигналы и сводки не теряются, а живут как отдельные сущности интерфейса.
- Есть понятный drill-down из сводки в первичные данные.
